# SheLink – AI-Driven Women Employment Platform (MVP)

## Problem
Millions of women face unemployment due to social constraints, lack of flexibility, and accessibility barriers.

## Solution
SheLink is an AI-driven, accessibility-first platform that connects women to flexible jobs through guided skilling and trusted recruitment.

## MVP Features
- Accessibility-first onboarding
- AI-based skill & job matching
- Voice-enabled interaction
- Flexible job discovery (WFH, part-time)
- Recruiter dashboard & verified placement

## Tech Stack
- Firebase Authentication
- Firestore
- Cloud Functions
- Cloud Storage
- Firebase Hosting
- Gemini AI
- Speech-to-Text / Text-to-Speech
- Google Analytics
